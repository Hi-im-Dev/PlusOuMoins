## Getting Started

To build the project, first make sure your ENV is correctly setup for a Qt compilation (qmake and make in PATH).

Then run theses commands:
```
$> mkdir build

$> cd build

$> qmake ..

$> make ..
```
And run the associated makefile for each project part (server / client).
